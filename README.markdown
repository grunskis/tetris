TETRIS
======

Yet another Tetris clone!

## Gameplay

Each cleared line gives you 1 point. Only the highest score is saved.

After every 5 cleared lines you enter a new level. 

## Controls

* Left      - LEFT ARROW
* Right     - RIGHT ARROW
* Rotate    - CONTROL
* Soft drop - DOWN ARROW
* Hard drop - UP ARROW
* Pause     - P
